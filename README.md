# 🌦️ Weather Dataset Analysis with Python

A beginner-friendly Data Analysis project built using **Python** and **Pandas**. This project explores a real-world weather dataset and demonstrates how to analyze, inspect, and extract meaningful insights from data.

## 📌 Project Overview

In this project, I performed Exploratory Data Analysis (EDA) on a weather dataset containing hourly weather observations.

The goal was to understand the dataset structure, analyze weather conditions, and practice essential Pandas operations used in real-world data analysis workflows.

---

## 📊 Dataset Information

The dataset contains:

* 8,784 weather observations
* Temperature data
* Dew Point Temperature
* Relative Humidity
* Wind Speed
* Visibility
* Atmospheric Pressure
* Weather Conditions

## 🔍 Key Insights

* The dataset contains 50 unique weather conditions.
* Mainly Clear and Mostly Cloudy were the most frequent weather conditions.
* No missing values were found in the dataset.
* The dataset includes both numerical and categorical features useful for analysis.
